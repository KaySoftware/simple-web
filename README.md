# Simple-web
A simple web page

## Used
<ul>
    <li>HTML5</li>
    <li>CSS3 (Tailwind CSS)</li>
    <li>Javascript (Typescript, Webpack)</li>
</ul>

## Instructions

<p>First use the command</p>

>npm install

<p>For run in develop mode</p>

>npm run start

<p>Build for prod</p>

>npm run build

<p>All files generated will be in <b>dist</b> folder</p>
